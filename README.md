# Collection de Bruno para testing de API REST

Para esta collection se utiliza la API REST publica de la siguiente web:

 => https://restful-api.dev/


La collection es un ejemplo que se utilizará para:

1. Hacer testing de una API REST.
2. Documentar los test/request para toda la API
3. Agregar scrpts de test a cada request.
4. Crear escenarios "ficticios" de pruebas.
5. Ejecución en batch (runner) de los casos de prueba.
